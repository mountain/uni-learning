# Q-Learning：第一遍（原初还原）

## 书目信息

- 标题：Q-Learning
- 作者：Christopher J. C. H. Watkins、Peter Dayan
- 年份：1992
- 发表：Machine Learning, Vol.8, pp.279-292

## 第一遍八项输出

### 1) 原始问题

引言强调无需环境模型即可进行在线策略改进。

### 2) 问题重要性

论文提出无模型时序决策学习算法 Q-learning，在延迟回报环境中通过值迭代思想实现渐进最优策略逼近。

### 3) 更大心理地图

作者将问题组织为：马尔可夫决策问题设定；Q 值更新规则；收敛性质与条件。

### 4) 同一整体中的关键对象

- 马尔可夫决策问题设定
- Q 值更新规则
- 收敛性质与条件

### 5) 核心概念（3–7）

1. 马尔可夫决策问题设定
2. Q 值更新规则
3. 收敛性质与条件
4. open computation / limit / anytime
5. 问题建模
6. 边界条件
7. 论证结构

### 6) 方法承诺

作者在 open computation / limit / anytime 路线内采用结构化建模与分层推进的方法承诺。

### 7) 主动切除的边界

- 优先处理文献中明确定义的问题对象，不外推到未定义任务
- 区分文献内部可证明主张与外部解释性扩展
- 重点防止误读：把收敛结论外推到任意函数逼近与非平稳环境；将 Q-learning 与规划方法混同而忽略模型无关设定

### 8) 一句压缩：该文献开辟了何种问题空间

该文献开辟了一个围绕“open computation / limit / anytime”组织问题并以结构化论证推进结论的问题空间。
