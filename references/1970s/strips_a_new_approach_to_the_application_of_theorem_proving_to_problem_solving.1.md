# STRIPS: A New Approach to the Application of Theorem Proving to Problem Solving：第一遍（原初还原）

## 书目信息

- 标题：STRIPS: A New Approach to the Application of Theorem Proving to Problem Solving
- 作者：Richard Fikes、Nils Nilsson
- 年份：1971
- 发表：Artificial Intelligence, Vol.2, pp.189-208

## 第一遍八项输出

### 1) 原始问题

引言强调将一般定理证明技术转化为可操作的规划流程。

### 2) 问题重要性

论文提出 STRIPS 规划框架，以状态表示、动作前提与效果规则组织问题求解，建立经典符号规划的可计算路径。

### 3) 更大心理地图

作者将问题组织为：问题表示与世界状态建模；动作算子与前提-效果机制；定理证明与规划搜索结合；系统实现与示例任务。

### 4) 同一整体中的关键对象

- 问题表示与世界状态建模
- 动作算子与前提-效果机制
- 定理证明与规划搜索结合
- 系统实现与示例任务

### 5) 核心概念（3–7）

1. 问题表示与世界状态建模
2. 动作算子与前提-效果机制
3. 定理证明与规划搜索结合
4. 系统实现与示例任务
5. cybernetics / automata / Dartmouth
6. 问题建模
7. 边界条件

### 6) 方法承诺

作者在 cybernetics / automata / Dartmouth 路线内采用结构化建模与分层推进的方法承诺。

### 7) 主动切除的边界

- 优先处理文献中明确定义的问题对象，不外推到未定义任务
- 区分文献内部可证明主张与外部解释性扩展
- 重点防止误读：把 STRIPS 的封闭世界假设忽略后直接外推到开放环境；将符号规划等同于一般智能而忽略其任务结构前提

### 8) 一句压缩：该文献开辟了何种问题空间

该文献开辟了一个围绕“cybernetics / automata / Dartmouth”组织问题并以结构化论证推进结论的问题空间。
