# Language Models are Few-Shot Learners：第一遍（原初还原）

## 书目信息

- 标题：Language Models are Few-Shot Learners
- 作者：Tom B. Brown et al.
- 年份：2020
- 发表：NeurIPS

## 第一遍八项输出

### 1) 原始问题

引言强调通过规模化预训练获得任务条件下快速适应能力。

### 2) 问题重要性

论文展示大规模语言模型在少样本与零样本设定下的跨任务能力，推动 in-context learning 成为核心研究方向。

### 3) 更大心理地图

作者将问题组织为：大规模模型训练设定；few-shot 评估框架；多任务结果与局限分析。

### 4) 同一整体中的关键对象

- 大规模模型训练设定
- few-shot 评估框架
- 多任务结果与局限分析

### 5) 核心概念（3–7）

1. 大规模模型训练设定
2. few-shot 评估框架
3. 多任务结果与局限分析
4. LM / agent / AGI
5. 问题建模
6. 边界条件
7. 论证结构

### 6) 方法承诺

作者在 LM / agent / AGI 路线内采用结构化建模与分层推进的方法承诺。

### 7) 主动切除的边界

- 优先处理文献中明确定义的问题对象，不外推到未定义任务
- 区分文献内部可证明主张与外部解释性扩展
- 重点防止误读：将 few-shot 成果误读为通用可靠推理能力；忽略评测协议、数据污染与提示敏感性影响

### 8) 一句压缩：该文献开辟了何种问题空间

该文献开辟了一个围绕“LM / agent / AGI”组织问题并以结构化论证推进结论的问题空间。
